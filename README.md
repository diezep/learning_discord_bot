# Searcher Discord Bot
Bot created to learn new things as a developer.

## Instructions
- Install discord.py pip package
- Create .env file with DISCORD_TOKEN variable.
- Run main.py

## Searchers
- Google [_google]
- Stardew Valley [_stardew, _valley, _stardewv, _stardewvalley]
- Terraria [_terraria]
